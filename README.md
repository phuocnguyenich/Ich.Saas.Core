# Ich.Saas.Core
SaaS (Software as a Service) solution that is multi-tenant, multi-timezone,
multi-language, multi-locale, and multi-currency.

## Prerequisites
- .NET Core 3.1
- Visual Studio 2022
- SQL Server

## In this repository
- ASP.NET Core MVC
- Entity Framework Core
- AspNetCore Identity
- Unit of Work
- Memory cache
- Lazy Load
- AutoMapper

## Steps to run
- Simply open the solution, compile, and run :))
